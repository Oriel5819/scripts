# CREATE AUTO COMMAND IN LINUX

1. create a ".sh" file 
2. add "#!/bin/bash" at the first line of the just created file
3. add some linux command in the file
4. change the mode of the file in order to be executable "chmod +x <filename>"
5. sudo copy the file in "/urs/local/bin" and give it a filename as the command

```bash
e.g.: sudo cp file_to_copy.sh /usr/local/bin/filename_as_command

filename_as_command // use this as command

```
